    2. [Ikke alt kan være kontstante, man trenger variabler for data som endres/oppdateres. du bruker const når verdien ikke skal endres, og let når den skal endre.]

  2. How do parentheses affect the arithmetic outcome of an expression?

    3. [Paranteser endrer rekkefølgen på hvordan uttrykket blir regnet ut, og det som er i paranteser endrer rekkefølgen på hvordan uttrykket blir regnet ut]

  3. How do we limit floating-point numbers to a given number of decimals?

    4. [toFixed() metoden kan brukes for å brgrense antall desimaler i et flyttall, den returnerer en string, slik at du kan kovertere det tilbake til ett nummer hvis det trengs]

  4. How can we compare two floating-point values to see if they are equal (is 12.199999 the same as 12.2)?

    5. [Tallene er ikke helt like, men for kritiske økonomiske beregninger bør du bruke skalerte tall, for eksempel håndtere pengeverdi som heltall i ører eller cent, i stedet for desimaltall i kroner eller dollar. Du kan sammenlikne floating-point verdiene ved å bruke toFixed() og så bruke en comparison operator]

  5. What do comparison operators return, and how can we check if they are FALSE or TRUE?

    6. [sammenligningsoperatorer returnerer en boolean verdi (true eller false) og vi kan sjekke om de er true eller false ved å bruke en if statament]

  6. How does the Math.random() function work, and what can it be used for?

    7. [Math.random() funksjonen returnerer et flyttall mellom 0 (inkludert) og 1 (ekskludert). man kan bruke math.floor() eller math.ceil() for å nå høyere eller lavere verdier. den kan brukes til å lage tilfeldige tall, f.eks i spill eller for å simulere tilfeldige hendelser]

  7. What is modulus (%), and when do we need this operator in mathematical operations?

    8. [modulus operatoren (%) returnerer resten av en divisjon mellom to tall, den er nyttig når vi trenger å finne ut om et tall er partall eller oddetall, eller når vi vil begrense et tall innenfor et bestemt område, f.eks i en sirkel eller en liste]
